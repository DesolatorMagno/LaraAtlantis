Extras
-------

- Dentro del archivo .env crear una variable **APP_THEME** con valor **dark** o **blue** para seleccionar entre tema obscuro o 
  claro (el claro viene por defecto).
- En caso de querer crear mas temas sean libres de modificar el archivo **config/theme.php** y jugar con lo que trae el tema.
- En el navegador ir a la seccion **/theme** para ver una seccion extra oculta con elementos de ejemplos para un CRUD completo
  que se para estos ejemplos se utiliza las vistas dentro de **theme/example01** y usando de template general del sistema
